The mechanism

Every agent run is a puzzle with a hidden key

Modern agents do not expose internal reasoning as a stable interface. Scotia reconstructs intent from the structured traces they cannot help but leave behind.

🧲

Intercept telemetry, not thought

Wrap any supported agent. Scotia tees stdout/stderr and parses tool calls, routing hints, and state changes.

⚙️

Normalize to a canonical algebra

Sort, deduplicate, and coalesce raw events into nine semantic event types that reconstruct execution as state transitions.

🔮

Synthesize post-hoc rationale

Generate Markdown summaries and Graphviz DOT graphs that explain what happened without inventing hidden reasoning.

🔒

Commit to Git as evidence

Each run becomes a deterministic artifact under scotia-log/: JSON event log, human summary, and action graph.

Documentation

What you can build with the ledger

🔍 Debugger for agent cognition

Reconstruct "thought" from action graphs, retries, and tool selection patterns rather than hidden reasoning.

🧪 Regression test generator

Turn any captured run into deterministic assertions: expected tools, model routes, state changes, and error counts.

📋 Compliance audit layer

Every code change driven by an LLM is accompanied by a Git-storable decision graph and diffable event log.

🧠 Cross-model orchestration memory

Track why Groq, Ollama, OpenAI, or local models were chosen for each stage and how fallback chains behaved.
